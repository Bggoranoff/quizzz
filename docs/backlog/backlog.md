## Question page

---

| Must have | Should have | Could have | Won’t have |
| --- | --- | --- | --- |
| As a player, I want to be able to see one of 20 selected multiple-choice questions and its answers, so that I can think about the question. | As a player, I want to be able to be able to use the “remove incorrect answer” joker, so that I have a better chance to answer the question correctly. | As a player, I want to be able to see an image near each question, so that the game is more interactive. |  |
| As a player, I want to be able to click on a question answer, so that I can answer the question | As a player in a multiplayer game, I want to be able to use the “decrease time for everyone else” joker, so that I lower the chances of others of answering the question correctly. | As a player in a multiplayer game, I want to be able to use more jokers (functionality that boosts my chance to answer a question correctly or get more points than the other players), so that the game is more competitive. |  |
| As a player, I want to be able to see a dynamic time countdown indicating the remaining time for the given question so that I know how much time I have to answer the question. | As a player, I want to be able to use the “double points” joker, so that I can get more points by answering a question correctly. | As a player in a multiplayer game, I want to be kicked from the game if I don’t answer 3 consecutive questions so that others’ gameplay is not slowed down if I am consistently inactive. |  |
| As a player, I want my number of correct answers to be kept on the server, so that I can see a ranking later. | As a player, I want my number of correct answers to be converted to a certain amount of points, so that the ranking is more intuitive for me. | As a player, I want to be able to see a round timer animation near the time countdown, so that the game is more interactive. |  |
| As a player, I want to be redirected to the result page after I answer the final question, so that I can see my result and start a new game. | As a player, I want to be able to react to a question and my reaction to be kept on the server, so that there is an emotional part to the game. |  |  |
